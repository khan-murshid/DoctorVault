# Doctor Vault

Doctor Vault is a healthcare-focused software project designed to explore the digital organization and management of medical information.

## Features

* Digital record organization
* User-friendly interface
* Secure information handling
* Practical healthcare-focused workflows

## Technologies Used

* TypeScript
* React
* Vite
